# Froot Viral Video Playbook

## Tools
- **Screen Studio** — screen recording with auto-zoom/cursor effects
- **CapCut** — editing, captions, trending audio

---

## Video 1: The "Wrong Size" Reveal (your main viral play)

### Record (Screen Studio)
- Open froot.fit in browser — use phone or Screen Studio's device frame (must be vertical)
- Record the full measurement flow → result reveal
- Do it twice: once fast (final cut), once slow (b-roll)

### Hook (first 2 seconds)
Pick one:
- "Victoria's Secret has been lying to you"
- "You're not a 34B"
- "I built a tool that proves you're wearing the wrong bra size"

Record yourself saying it (selfie cam, natural light) or use as text overlay if no face.

### Edit (CapCut)
1. Import hook clip (or black screen + text)
2. Cut to Screen Studio recording of froot flow
3. Speed ramp the measurement input steps (1.5-2x)
4. Slow down the result reveal moment
5. Add auto-captions (bold white, black outline)
6. Add sound — trending audio or moody beat from CapCut's trending tab
7. **Total length: 15-30 seconds max**

### Post
- **TikTok first** (that's where it spreads), then cross-post to Reels and Shorts
- Caption: "I built this. link in bio. you're welcome."
- Hashtags: `#brafitting #bratok #brasizecalculator #frootfit #victoriassecret`
- Post between 6-9pm your timezone
